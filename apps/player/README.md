# Player app (`@repo/player`)

**Player-facing** casino shell. This is the **main public app** — not the admin panel.

- Deploy on your **public** domain (e.g. `www` or `app`).
- **Do not** ship staff routes or TailAdmin here.
- **Do not** import code from `apps/admin`.
- Shared styling only via `@repo/design-tokens`.

See **`docs/separate-admin-and-player.md`**.
